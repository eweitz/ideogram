Overview | Previous | Next | Source

A way to get a cursor or click handler for your ideogram. If you set the `cursorPosition` parameter, the rotation will be handled via double click.

Selected position on human chromosome 1: 
  var config = {
    container: '.ideogram-container',
    organism: 'human',
    chromosome: '1',
    chrHeight: 900,
    resolution: 550,
    orientation: 'horizontal',

    cursorPosition: 110673777; // set a default position
    onCursorMove: writeCurrentPosition // event handler when position is clicked
  };
